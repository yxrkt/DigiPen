---------
 Buttons
---------

[1]Load Image  - loads a .bmp, .jpg, or .tif image
               - filter starts on .bmp files

[2]Restore     - restore originally loaded image

[3]Get Objects - labels image using auto-threshold
               - info displayed in status bar

[4]Apply Open  - applies open to binary image
               - mask selected with radio buttons


-------
 Other
-------

Region Growing - click on image
               - region is selected with 8-way
                 connectivity
               - 'binary' unchecked fills the
                 region with green
               - 'binary' checked turns image
                 black and makes region white
